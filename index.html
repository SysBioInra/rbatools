RBAtools is a programming interface for Resource Balance Analysis (RBA) models based on the RBApy package. It includes methods to solve, modify, and analyse RBA models and to export model information and simulation results into various formats.

RBA models have already been developed for:

  • Bacillus subtilis 168 (wild type)
  • Escherichia coli K-12 (wild type)
  • CO2-fixing Escherichia coli K-12 (engineered strain)
